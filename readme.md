# helpman-hsp3
HSPスクリプトエディタのF1キー機能をAtomで再現します。

## インストール方法
コマンドプロンプトで`apm install helpman-hsp3`を実行してください。

## 使用方法
エディタの言語モードがHSPの状態で、調べたいキーワード上にキャレットカーソルを移動させて`F1`キーを押してください。

## 設定方法
パッケージ詳細からSetting項目で、helpman.exeファイルの絶対パスを指定してください。
hsphelpフォルダにhelpman.exeが無ければ、HDL作者様からダウンロードして配置してください。
